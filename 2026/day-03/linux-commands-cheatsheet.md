# Linux Commands Cheat Sheet
---

## File & Directory Management
- `ls` – List files & directories  
- `cd <dir>` – Change directory  
- `pwd` – Show current directory path  
- `cp <src> <dst>` – Copy file  
- `cp -r <src_dir> <dst_dir>` – Copy directory recursively  
- `mv <src> <dst>` – Move or rename file/directory  
- `rm <file>` – Remove file  
- `rm -r <dir>` – Remove directory recursively  
- `mkdir <dir>` – Create a directory  
- `touch <file>` – Create empty file or update timestamp

---

## File Viewing & Editing
- `cat <file>` – View file content  
- `less <file>` – View file content page by page  
- `head <file>` – Show first 10 lines  
- `tail <file>` – Show last 10 lines  
- `nano <file>` – Edit file in terminal  
- `vim <file>` – Edit file using Vim

---

## Process Management
- `ps aux` – List running processes  
- `top` – Interactive process viewer  
- `kill <PID>` – Terminate process by PID  
- `Ctrl + Z` – Stop foreground process  

---

## Networking
- `ping <host>` – Check host connectivity  
- `ip addr` – Show network interfaces & IPs  
- `curl <URL>` – Fetch content from URL  
- `dig <domain>` – Query DNS info  

---

## System Info & Disk
- `df -h` – Disk usage  
- `du -sh <dir>` – Size of directory  
- `free -h` – Memory usage  
- `uname -a` – Kernel & system info  
- `uptime` – System uptime  

---

## File Permissions
- `chmod 755 <file>` – Change file permissions  
- `chown user:group <file>` – Change file owner
